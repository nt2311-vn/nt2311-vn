### 🧭 About

I work across the stack but mostly live on the systems / backend side: agent
sandboxes, microVMs, LLM gateways, RAG, and the boring-but-critical infra glue
that keeps them running. I’m currently learning **Zig** and going deeper on
**Rust** while shipping production Go.

- 🧠  Currently exploring **microVM hypervisors**, **WASI runtimes**, and **on-device inference**.
- 🛠️  Tooling-obsessed: `mise` for toolchains, `just` for task running, Neovim for everything else.
- 🌏  Based in 🇻🇳 Vietnam · open to collaboration on Rust / Zig / Go infra projects.
